Linz

Welcome to Linz. A CMS of another kind.

Linz is not quite a CMS, but a powerful framework for creating administration interfaces. Linz is the framework of choice when the administration interface is the website itself.

If you're looking for a CMS, try something else like KeystoneJS. If you're looking for a highly customisable administration interface, that is also a treat to use, then you're in the right place.

Features

Express and MongoDB

Linz is backed with Express and MongoDB. If you're familiar with these projects, then you'll feel at home using Linz. Linz has been designed to keep out of the way unless you need it.

Auto-generated UI

Linz provides powerful DSLs to express database models and custom administration interfaces. These DSLs are turned into beautiful HTML, JavaScript and CSS to make it quick and easy to manage your data. If you don't like the default output, use an override to customise it.

Only when you need it

You can integrate as little, or as much of Linz as you like. Linz has purposely been designed so that you don't have to rework your entire codebase to get started.
